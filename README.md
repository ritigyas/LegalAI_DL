# ⚖️ Legal Intelligence Engine

An AI-powered system to analyze legal queries and documents, providing structured legal insights using IRAC methodology.

---

## 🚀 Features

- 📄 Chat with PDF documents  
- 🔍 Legal query analysis  
- ⚖️ Structured output:
  - Legal Issue  
  - Jurisdiction  
  - Key Facts  
  - Precedents  
  - IRAC Analysis  
- 🌐 Case law retrieval  
- 🤖 AI-based reasoning (Gemini)

---

## 🧠 Tech Stack

- Streamlit (UI)  
- Python (Backend)  
- pdfplumber (PDF parsing)  
- BeautifulSoup (Web scraping)  
- Google Gemini API (LLM)

---
